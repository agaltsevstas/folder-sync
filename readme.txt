Автоматическое монтирование сетевой папки при загрузке системы (ввод через терминал, пункты 1-5 выполнять только один раз)

1) Для работы c сетевыми дисками SMB/CIFS (Windows или Samba) требуется утилита cifs-utils. Для этого нужно ввести команду в терминале:
   sudo apt-get install cifs-utils

2) Далее требуется создать директорию, куда монтируется расшаренная сетевая папка. Для этого нужно ввести команду в терминале:
   mkdir your_directory

3) Требуется создать файл, куда можно вписать информацию о пользователе (логин, пароль, домен/рабочая группа). Для этого нужно ввести команду в терминале:
   sudo nano /root/.smbuser
   Туда вписывается:
      username=user
      password=password
      domain=nami

4) Для безопасности требуется ограничить доступ к файлу данными. Для этого нужно ввести команду в терминале:
   sudo chmod 400 /root/.smbuser

5) Требуется отредактировать файл /etc/fstab. Для этого нужно ввести команду в терминале:
   sudo nano /etc/fstab
   Туда вписывается:
      //192.168.9.170/videoai/your_directory /your_path/your_directory cifs credentials=/root/.smbuser,uid=1000,gid=1000 0 0

6) Запуск программы. Для этого нужно ввести команду в терминале:
   python3 folder_synchronization.py
