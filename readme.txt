Автоматическое монтирование сетевой папки при загрузки системы (ввод через терминал и делаем только один раз)

1) Для работы c сетевыми дисками SMB/CIFS (Windows или Samba) нам понадобиться утилита cifs-utils. Для её установки открываем терминал и набираем следующую команду:
   sudo apt-get install cifs-utils

2) Далее потребуется создать директорию, куда мы хотим примонтировать нашу расшаренную сетевую папку. Для этого давайте наберем вот такую команду:
   mkdir your_directory

3) Для этого создадим файл, куда мы сможем вписать информацию о пользователе(логин, пароль, домен/рабочая группа)
   sudo nano /root/.smbuser
   Пишем туда:
   username=user
   password=password
   domain=nami

4) Для безопасности ограничим доступ к файлу с нашими данными:
   sudo chmod 400 /root/.smbuser

5) Редактируем файл /etc/fstab 
   sudo nano /etc/fstab

6) Далее набираем команду для монтирования нашей сетевой папки.
   //192.168.9.170/videoai/your_directory /your_path/your_directory cifs credentials=/root/.smbuser,uid=1000,gid=1000 0 0

7) Можно запускать нашу программу
   python3 folder_synchronization.py
